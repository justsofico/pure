# Markdown Notes Organizer

A lightweight Python application for organizing Markdown notes.

## Overview

Markdown Notes Organizer is a simple console application that manages Markdown notes, groups them by tags, and exports a formatted summary.

The project demonstrates object-oriented programming, modular architecture, and basic text processing.

## Features

- Store Markdown notes
- Group notes by tags
- Count notes
- Export formatted summaries
- Clean project structure
